## 테이블 설계

> 작성일 : `2022-08-06`

- PK : 기본 키 > Index
- FK : 참조 키 > Index
- UN : 고유 키
- NN : NOT NULL
- AI : Auto Increment

### 유저

| 키        | 타입       | 범위          | DB 범위       | 제약 조건 |
| :-------- | :-------- | :------------ | :------------ | :------- |
| user_id   | number    |               | INTEGER       | NN PK  AI |
| nickname  | string    | 2 <= - <= 15  | VARCHAR(15)   | NN UN     |
| password  | string    | 4 <= - <= 20  | `VARCHAR(255)` | NN       |
